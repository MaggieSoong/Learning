# ML Interview Complete Study Guide

An interactive, single-file HTML study guide covering the core topics for machine learning, GenAI/LLM, and NLP interviews.

## What's Inside

Four tabbed sections: **Machine Learning**, **ML Basics**, **GenAI / LLMs**, and **NLP** with expandable topic cards, inline SVG diagrams, code snippets, interview questions, and a progress tracker per section.

## How to Use

Open `study-guide.html` in any browser. Click a tab to switch sections, then click any card to expand it. Your progress (cards opened) is tracked per tab at the top of each section.

## Topics at a Glance

- **Machine Learning** (12 topics): Supervised/unsupervised learning, bias-variance, algorithms, cross-validation, metrics, tuning, ensembles, interpretability, MLOps, and case studies (PD models, supply chain, ESG)
- **ML Basics** (6 topics): What is ML, data fundamentals, curse of dimensionality, imbalanced classes, feature engineering, loss functions
- **GenAI / LLMs** (9 topics): Transformers, pre-training, fine-tuning vs RAG, prompt engineering, agents, evaluation, hallucinations, AI readiness, chatbot validation
- **NLP** (8 topics): Word embeddings, RNNs/LSTMs, text classification, NER, QA, translation, summarisation, and a credit rating NLP case study
